# User Management Frontend

This directory contains the React frontend codebase for the User Management application.

For complete project-wide documentation, including architecture, installation, and deployment details, please refer to the main [Root README.md](../README.md).

## 🚀 Quick Start

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Run development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

4. **Lint code**:
   ```bash
   npm run lint
   ```
